# 🥊 AI Arena

Два ИИ сходятся в риторическом бою по заданной теме — **конфликт, ставки и судья**, который выносит вердикт.

**▶ Живое демо:** https://panarini.github.io/ai-arena/

Задаёшь тему и ставку, выбираешь двух бойцов из шести характеров и число раундов — они пикируются, судья сводит раунды по очкам и объявляет победителя. Можно переиграть, сменить расклад и скопировать бой, чтобы показать.

## Слои

- **v0 (сейчас):** «мозг» бойцов — шаблонный движок (библиотека риторических ходов + подстановка твоей темы). Видимый работающий формат без бэкенда и ключей.
- **Следующий слой:** живой ИИ по углам — реальные LLM-реплики через бэкенд.
